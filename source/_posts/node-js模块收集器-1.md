## 类似于ipython的工具

Read–eval–print loop,简称Repl
nodejs 除了自带的 node，也有一些第三方 repl 增强。nesh 就是其中很优秀的一个。不得不说，node 世界最近发展迅速，开发者热情高涨，插件、库层出不穷。nesh plugins

npm install -g nesh
# Run nesh
nesh
# Run nesh with CoffeeScript
nesh -c
# Run nesh with ES6 through Babel
nesh -b
